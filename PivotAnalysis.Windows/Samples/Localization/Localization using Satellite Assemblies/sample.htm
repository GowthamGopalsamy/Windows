This sample illustrates the Localization of PivotGrid Control and this sample is localized with German culture(de-DE).

Steps for Performing Localization:

The Strings in could be Localized using satellite Assemblies. A text File containing the key and the corresponding string is added. The resource File is then created from it using the following command in Visual Studio Command Promt.

resgen Location\SR.txt Location\SR.resources

The resource File could be altered by the user as per his needs. From the resource File, the satellite dlls are created using the following command:

al /t:lib /culture:de-DE /out:Location\Syncfusion.PivotAnalysis.Windows.resources.dll /v:1.1.0.0 /delay+ /keyf:F:\\sf.publicsnk /embed:F:\SR.Resources

The obtained resource file is then placed in the following location:


< Location of Form>\bin\debug\de-DE (In case of German Culture)


The Program could be ran to see the changes made in the Resource File.
