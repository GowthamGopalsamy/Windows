This sample illustrates the localization support available in pivot grid, in which the pivot grid control is localized using German (de-DE) culture.

Steps to Perform Localization:

  • The strings in pivot grid could be localized using satellite assemblies.

  • The text file containing the key and the corresponding string needs to be added.

  • Then the resource file is created from it using the following command in Visual Studio Command Prompt.


    • resgen Location\SR.txt Location\SR.resources

  • The resource file can be altered as the user wants. From the resource file, the satellite dlls are created using the following command:


    • al /t:lib /culture:de-DE /out:Location\Syncfusion.PivotAnalysis.Windows.resources.dll /v:1.1.0.0 /delay+ /keyf:F:\\sf.publicsnk /embed:F:\SR.Resources

  • Then the obtained resource file needs to be placed in the following location:


    • <Location>\bin\debug\de-DE (In case of german culture)

  • Then the program could be ran to see the changes made in the resource file.
