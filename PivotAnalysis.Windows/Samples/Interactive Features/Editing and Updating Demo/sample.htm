This sample illustrates the capability of dynamically editing and updating of PivotGrid data in runtime and updating it in datasource too.

  • The EnableValueEditing check box allows you to enable/disable editing option in the PivotGrid control.

  • The Enable Updating check box allows you to enable/disable updates in the PivotGrid control.

  • The Perform Updates check box will perform updates to the pivotgrid control using Timer.

  • The EditingTotal Cell checkbox will enable/disable the editing option of the Total cell value.

  • The Hide Expander checkbox will hide/show the expander icon

  • The Add buttons will add rows to the top, middle, or bottom of the grid respectively.
