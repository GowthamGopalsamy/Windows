This sample illustrates the pivot grid that is customized and displayed in a 3D structural format.

Features:

The following functionalities are implemented in row pivots only mode:

  • Sorting:

    • The pivot calculation item has been provided with the sorting support.

    The sorting operation can be enabled or disabled by using the AllowSort property.

  • Filtering:

    • The pivot calculation item has been provided with the filtering support, whereas the pivot row items are not supported with filters.

    The filtering operation can be enabled or disabled by using the AllowFilter property.

  • Pivot Value Chooser:

    • Pivot value chooser is a customized dialog which showcases the available pivot calculation items in a context menu.

    From the available pivot calculation items, the user can choose the required pivot items.
