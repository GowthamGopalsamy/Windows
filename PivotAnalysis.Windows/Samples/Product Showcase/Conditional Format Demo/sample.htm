This demo illustrates conditional formatting support in the PivotGrid control.


The panel on the right enables you to switch between different rule types, apply conditions, and set criteria for formatting in Pivot Grid value cells.

Here is a short code Snippet: 



NewRuleConditionalFormat newRule1 = new NewRuleConditionalFormat();
newRule1.FormatStyle = FormatStyle.IconSets;
newRule1.SummaryElement = "Quantity";            
newRule1.Image = BitMapImageName.FourArrowsGray;            
newRule1.RuleType = RuleType.FormatAllCellsBasedOnTheirValues;            
PivotGridNewRuleConditionalFormat newRuleFormat1 = new PivotGridNewRuleConditionalFormat();
newRuleFormat1.NewRuleCollections.Add(newRule1); this.pivotGridControl1.TableControl.NewRuleConditionalFormat.Add(newRuleFormat1);
