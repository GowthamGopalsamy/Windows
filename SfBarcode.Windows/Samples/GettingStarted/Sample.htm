This sample demonstrates how data can be encoded as one or two dimensional Barcode.

The following types of Barcodes are supported:

1D Barcodes

  • UpcBarcode
  • Code39
  • Code39Extended
  • Code11
  • Codabar
  • Code32
  • Code93
  • Code93Extended
  • Code128
    • Code Set A (or Chars Set A)
    • Code Set B (or Chars Set B)
    • Code Set C (or Chars Set C)
2D Barcodes
  • DataMatrix
  • QRBarcode
