This sample illustrates how to add the Excel-like cell comments to the grid cells.

The following code is used to set the comment tip for particular column,

  
// Add the comment tip text for particular column
this.gridGroupingControl1.TableDescriptor.Columns["Column_Name"].Appearance.AnyRecordFieldCell.CommentTip.
CommentText = "Comment Text";

// Set the Image for the comment tip window.
this.gridGroupingControl1.TableDescriptor.Columns["Column_Name"].Appearance.AnyRecordFieldCell.CommentTip.CommentImage = 
Image.FromFile(@"../../CommentTipImage.png");
