This sample illustrates how to apply style settings dynamically in a GridGroupingControl.

Provide formatting at run time using the QueryCellStyleInfo event handler.

QueryCellStyleInfo provides the GridStyleInfo object for a given cell on demand.

The following code snippet illustrates how to apply the styles for specific cell

 private void gridGroupingControl1_QueryCellStyleInfo(object sender, Syncfusion.Windows.Forms.Grid.Grouping.GridTableCellStyleInfoEventArgs e)
 {
     if(e.TableCellIdentity.TableCellType == GridTableCellType.RecordFieldCell)
     {
         if(e.TableCellIdentity.ColIndex %2 ==0)
         {
             e.Style.Font.FontStyle=FontStyle.Bold;
             e.Style.BackColor=Color.FromArgb(255, 187, 111);
         }
         else
         {
             e.Style.TextColor = Color.White;
             e.Style.BackColor=Color.FromArgb(55, 91, 114);
         }
     }
 }
