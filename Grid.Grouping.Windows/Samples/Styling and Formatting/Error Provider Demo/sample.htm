This sample illustrates how to display the error icon and set custom error text in a GridGroupingControl.

  • ShowHeaderErroricon is used to show error icon on RowHeader.

                this.gridgroupingcontrol1.ShowRowHeaderErroricon = true;

  • ShowErroricon is used to show error icon in CurrentCell.

                this.gridgroupingcontrol1.TableControl.CurrentCell.ShowErroricon = true;

  • ShowErrorMessageBox is used to show the error Messagebox.

                this.gridgroupingcontrol1.TableControl.CurrentCell.ShowErrorMessageBox = true;

  • ValidationErrorText this text will display in the error MessageBox.
