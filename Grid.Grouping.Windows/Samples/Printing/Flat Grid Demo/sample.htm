This sample shows how to print a grid on a single page. It derives a GridPrintDocumentAdv class in order to handle the printing of multiple grids across various pages.

Printing:

  • Declare an object for the class GridPrintDocumentAdv with its constructor parameter as the grid-control object.

    Syncfusion.GridHelperClasses.GridPrintDocumentAdv pd = new Syncfusion.GridHelperClasses.GridPrintDocumentAdv(this.gridGroupingControl1);
