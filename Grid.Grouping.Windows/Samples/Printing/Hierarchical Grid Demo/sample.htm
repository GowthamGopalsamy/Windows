This sample illustrates how to print or preview the content of a Grid Grouping Control.

The following code shows how to print data from the data grid.

 GridPrintDocument pd = new GridPrintDocument(this.gridGroupingControl1.TableControl); 
 PrintDialog printDialog = new PrintDialog(); 
 printDialog.Document = pd;
 pd.DefaultPageSettings.Landscape = true;
 printDialog.ShowDialog();
