This sample illustrates how to create custom cell and register that cell in a Grouping control.

In this sample, the My Header cell is customized in the HeaderCell class.After defining MyHeaderCell, it is instantiated in the main form using the following code:

 MyHeaderCellModel header = new MyHeaderCellModel(this.gridGroupingControl1.TableModel); 
 this.gridGroupingControl1.TableModel.CellModels["My Column Header"] = header;
