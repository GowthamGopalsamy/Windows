This sample illustrates how to export the GridGroupingControl to Excel with various options.
This GridGroupingControl can be exported to Excel with styling(Backgroundcolor, Fonts, Borders), Hierarchy(Grouping, NestedTables) and so on.

  • XlsIO libraries are used to convert grid content to Excel. The dependent .dll files are Syncfusion.XlsIO.Base and Syncfusion.GridConverter.Windows. 

                    Syncfusion.GroupingGridExcelConverter.GridGroupingExcelConverterControl excelConverter = new
                Syncfusion.GroupingGridExcelConverter.GridGroupingExcelConverterControl();
                ExcelExportingOptions exportingOptions = new ExcelExportingOptions();
                excelConverter.ExportToExcel(this.gridGroupingControl1, 
                saveFileDialog.FileName, exportingOptions);
