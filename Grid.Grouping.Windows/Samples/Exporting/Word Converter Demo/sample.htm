This sample demonstrates the conversion of Grouping Grid data into a Word document.

Essential DocIO libraries are used to support the conversion of grid content into a Word file. The dependent .dll files are Syncfusion.DocIO.Base and Syncfusion.GridHelperClasses.Windows.

The following code used to covert the grid to word


GroupingGridWordConverter converter = new GroupingGridWordConverter(true, true);
converter.GroupingGridToWord("Sample.doc", gridGroupingControl1);
