This sample illustrates virtual-mode and without-counter-mode optimization in GridGroupingControl.

  • In this sample the automatic switching between the virtual mode and without-counter mode when you select the "Flat List VirtualMode" example.
  • The engine will be initialized using the virtual mode.
  • Once you sort by a column, the virtual mode will be turned off, but the without-counter mode will still continue to work.
  • Once you sort by a column, the counters will then be enabled.
