This sample illustrates the different modes of selection for the Grid Grouping Control cells.

The following code illustrate to enable the ListBoxSelection


this.gridGroupingControl1.TableOptions.ListBoxSelectionMode = SelectionMode.One;

Selection Types:

  • Row Only- Selects only the row

  • Column Only- Only the columns can be selected

  • Cell- Selects only the individual cell

  • Row and Column- Selects the column and row with respect to current cell

  • Default- Enables list box selection mode by default

  • None- Disables selecting cells
