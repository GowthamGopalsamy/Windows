This sample demonstrates localizing the GroupingGrid with the help of ILocalization provider to localize the data.

Localization:

	// To enable localization
	 LocalizationProvider.Provider = new Localizer();

Steps for Performing Localization:

The Strings in could be Localized using satellite Assemblies. A text File containing the key and the corresponding string is added. The resource File is then created from it using the following command in Visual Studio Command Promt. The SR.resources file could be obtained from the following location:

\\Windows\Grid.Windows\Samples\Localization Samples\Localization Demo\CS\Resources
The resource File could be altered by the user as per his needs. From the resource File, the satellite dlls are created using the following command:


al /t:lib /culture:de-DE /out:Location\Syncfusion.Grid.Grouping.Windows.resources.dll /v:1.1.0.0 /delay+ /keyf:F:\\sf.publicsnk /embed:F:\SR.Resources

The obtained resource file is then placed in the following location:


< Location of Form>\bin\debug\de-DE (In case of German Culture)


The Program could be ran to see the changes made in the Resource File.
