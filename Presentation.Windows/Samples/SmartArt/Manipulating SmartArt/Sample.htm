This sample demonstrates how to add and remove nodes in a SmartArt diagram and edit text of the nodes using Essential Presentation.

Features:

  • Supports to add and remove nodes in an existing SmartArt diagram.
  • Supports to edit text contents of the SmartArt nodes.
  • Supports to make a SmartArt node as assistant node.

Note1: This library supports SmartArt creation, manipulation & conversion to PDF/image for .pptx format alone.

Note2: The modifications in a SmartArt diagram (like add/remove nodes, modify position and size of nodes etc., which involve SmartArt layout changes) done by Essential Presentation are not supported in Image and PDF conversion. Whereas layout changes will be reflected properly in the generated PPTX file when opened using Microsoft PowerPoint.
