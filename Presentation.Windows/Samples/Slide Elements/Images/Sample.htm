Essential Presentation library has support for inserting images in a PowerPoint presentation.


This sample demonstrates how to insert an image in a PowerPoint slide.

Features:

  • Add new images to a PowerPoint slide.
  • Insert, delete, replace, and format pictures in a PowerPoint slide.

Note: This library supports creation, manipulation & conversion to PDF/image for .pptx format alone.
