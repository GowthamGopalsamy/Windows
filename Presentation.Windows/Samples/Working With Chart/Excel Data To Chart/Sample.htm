Essential Presentation library has support for creating a chart in a PowerPoint presentation from the data in an excel file. This will be helpful to import data from a database to PowerPoint presentation.


This sample demonstrates how to create a chart in a PowerPoint presentation, from the data in an existing excel file.

Features:

  • Create, edit and convert more than 80 chart types.
  • Create custom charts by adding combinations from different chart types.
  • Convert individual chart in a PowerPoint slide to image.
  • Modify, include and remove chart elements like axes, label, legend and chart title.
  • Edit color for chart elements like chart area and plot area of an existing chart.

Note: This library supports Chart creation, manipulation & conversion to PDF/image for .pptx format alone.
