This sample demonstrates how to create a chart in PowerPoint Presentation.

The Essential Presentation library provides support to below features.

Features:

  • Supports to create chart in PowerPoint Presentation
  • Supports to modify charts in existing PowerPoint Presentation
  • Supports to convert chart in a Presentation to image

Note: This library supports Chart creation, manipulation & conversion to PDF/image for .pptx format alone.
