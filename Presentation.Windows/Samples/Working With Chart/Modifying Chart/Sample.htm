This sample demonstrates how to customize the appearance of a chart in PowerPoint Presentation.

The Essential Presentation library provides support to below features

Features:

  • Supports to apply color for chart elements like chart area and plot area of an existing chart.
  • Supports to modify, include and remove chart elements like axes, label, legend and chart title.
  • Supports to customize the appearance of each chart elements.

Note: This library supports Chart creation, manipulation & conversion to PDF/image for .pptx format alone.
