Presentation library provides rich set of API to create & manipulate Microsoft PowerPoint (*.pptx alone) Presentations.This sample demonstrates how to do basic operations such as creating slides, shapes, paragraph & adding text.

Features:

  • Shapes & its formatting
  • Adding contents within the slide shapes
  • Font & paragraph formatting

Note: This feature is currently supported for .pptx format alone.

This library supports creation, manipulation & conversion to PDF/image for .pptx format.
