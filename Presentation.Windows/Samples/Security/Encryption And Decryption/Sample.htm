This sample illustrates the encryption and decryption of a PowerPoint Presentation using Essential Presentation.

Features:

  • Supports to encrypt a PowerPoint Presentation using agile encryption, similar to Office 2010 and 2013.
  • Supports to decrypt the PowerPoint Presentations encrypted using both agile and standard (Office 2007) encryption.

Note:The encrypted Presentations can only be read by Office 2007 SP2 compatibility pack or later Office versions.
