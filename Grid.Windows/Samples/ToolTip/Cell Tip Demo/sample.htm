This sample illustrates to specify a ToolTip for a range of cells or individual cells in GridControl.

The following code used to set the CellTipText for grid

Specific Column:

        this.gridControl1.ColStyles[4].CellTipText = "TipText for column 4";

Specific Row:

        this.gridControl1.RowStyles[5].CellTipText = "TipText for row 5";

Specfic cell

        this.gridControl1[2, 5].CellTipText = "TipText for cell 2,5";
