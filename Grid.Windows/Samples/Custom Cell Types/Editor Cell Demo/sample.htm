This sample illustrates the usage of different custom editor controls as celltypes in GridControl.

Integer Textbox Cell

    To have Integer TextBox cells, you should Register cell model as IntegerTextBox. Also you should set the cell type as IntegerTextBox.

            RegisterCellModel.GridCellType(this.gridControl1, CustomCellTypes.IntegerTextBox);
            
        gridControl1[rowIndex, colIndex].CellType = CustomCellTypes.IntegerTextBox.ToString();
