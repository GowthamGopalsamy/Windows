This sample illustrates the usage of different customized controls as celltypes in GridControl.

The following code used to add the CellModel of the entry

DragButton

        gridControl1.CellModels.Add("DragButton", new DragButtonCellModel(gridControl1.Model));
        gridControl1[4, 4].CellType = "DragButton";

EllipsisText

        gridControl1.CellModels.Add("EllipsisText", new EllipsisCellModel(gridControl1.Model));
        gridControl1[rowIndex, 8].CellType = "EllipsisText";
