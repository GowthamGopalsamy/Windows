This sample illustrates how to fit the row height and column width based on the content of the cells.

The following code used to retrieve the Correct size of the controls in grid cells and adjust respective RowHeights & ColWidths in order to make the cells to fit controls.

        gridControl1.RowHeights.ResizeToFit(GridRangeInfo.Table(), GridResizeToFitOptions.NoShrinkSize);
        gridControl1.ColWidths.ResizeToFit(GridRangeInfo.Table());
