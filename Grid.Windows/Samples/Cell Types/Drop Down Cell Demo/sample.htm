This sample illustrates the dropdown controls in the GridControl.

ComboBox celltype:

        gridControl1[rowIndex, 3].CellType = "ComboBox";
        gridControl1[rowIndex, 3].ChoiceList = items;

GridListControl celltype:

        gridControl1[rowIndex, 3].CellType = "GridListControl";

        gridControl1[rowIndex, 3].ChoiceList = items;
