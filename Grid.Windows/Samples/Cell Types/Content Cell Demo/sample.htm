This sample illustrates the flexibility of grid cells to make a cell as Static, Header, Progressbar and host the cell as a picturebox to display image.


The following code used to set the CellType as Static, Header, or Image to have the respective cells.

Static CellType:

        gridControl1[rowIndex,colIndex].CellType = "Static";

Header CellType:

        gridControl1[rowIndex, colIndex].CellType = GridCellTypeName.Header;

Image CellType

	       gridControl1[rowIndex, colIndex].CellType = GridCellTypeName.Image;
        gridControl1[rowIndex, colIndex].ImageList = imageList;
        gridControl1[rowIndex, colIndex].ImageIndex = 2;
