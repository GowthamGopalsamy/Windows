This sample demonstrates how a PDF document can be protected with user and owner passwords. Essential PDF has support for 40-, 128- and 256-bit encryption and the ability to restrict the following permissions:

  • Allow EditContent
  • Allow CopyContent
  • Allow EditAnnotations
  • Allow AccessibilityCopyContent
  • Allow AssembleDocument
  • Allow Print
  • Allow FullQualityPrint
