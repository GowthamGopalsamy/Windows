Essential PDF supports drawing text in a PDF document, which can flow to multiple pages. Text can be drawn using predefined fonts, CJK fonts, or true-type fonts.

  • The PdfTextElement class is used for drawing multiple pages of text into a PDF document.
  • The Layout property of the PdfLayoutFormat class helps in breaking the text into multiple pages.
  • Unicode text is also supported by this method.
