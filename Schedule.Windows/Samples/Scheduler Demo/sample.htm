This sample illustrates how to use the schedule control in a form and it also demonstrates various properties and settings of the schedule control.

Schedule Control:

  • The Navigation Calendar that appears by default to the left of a schedule control can be sized to show more months.
  • There is a context menu that will allow you to switch the types of views and also edit data in the schedule control.
Enabling to choose the Data Source.

The Scheduler sample enables us to provide the choose the desired data source. In the above scenarios, the data source is ScheduleProvider. We can also have "DataBase" as a data source for the Schedule control and manipulate the data with it.
