This sample illustrates how to navigate between bookmarks in a Word document using Essential DocIO. Bookmark navigation lets the user to switch the control or cursor to a specific bookmark and let the user to modify, edit or delete the contents on it.

Below is the illustration of the output document.

Features :

  • Supports navigation using bookmarks in a Word document.
  • Supports appending or deleting contents to an existing bookmark.
  • Supports Bookmark content replacement among tables, sections and paragraphs.
