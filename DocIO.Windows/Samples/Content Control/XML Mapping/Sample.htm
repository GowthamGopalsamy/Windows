This sample illustrates mapping custom XML parts to content controls in the Word document using Essential DocIO.

Note: Content controls are supported in DOCX file formats.

Below is the illustration of the output document.

Features :

  • Supports to create content control in Word document.
  • Supports to edit an existing content control in Word document.
  • Supports for mapping custom XML parts to content controls.
  • Supports for converting Word document with content controls to PDF.
