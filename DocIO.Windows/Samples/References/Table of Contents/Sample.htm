This sample showcases how to insert and update the Table of Contents (TOC) in a Word document using Essential DocIO. TOC is field which acts as an index to the Word document.

Features :

  • Supports inserting TOC into a Word document.
  • Supports inserting TOC based on default heading or custom styles.
  • Supports including page numbers and align-to-right options.
  • Supports Hyperlinks from TOC to navigate the document.
  • Supports setting of outline levels.
  • Supports updating Table of Contents.
