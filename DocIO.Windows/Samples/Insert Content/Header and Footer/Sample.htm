This sample illustrates inserting headers and footers into the Word document with all possible header and footer types like first page header and footer, different odd and even page header and footer using Essential DocIO.

Below are the illustrations of the output document demonstrating headers and footers.

Features :

  • Insert headers and footers to include text, images, tables, page numbers, dates, document title, file name and author name.
  • Add page numbers in formats of Arabic, Roman Upper and Roman Lower.
  • Restart page numbers for any section and change starting page number for each section.
  • Supports first page, odd page, even page header and footer.
