This sample illustrates inserting Bookmarks into the Word document using Essential DocIO. Bookmarks are used for reference purpose. A particular item like paragraph, table, etc., could be marked with a bookmark and it can be accessed by its name. Once after selecting a bookmark, the user can edit, modify or delete the contents in that bookmark.

Below is the illustration of the output document.

Features :

  • Identify a location or a selection of text named and saved for future reference.
  • Supports for creation of bookmarks.
  • Supports easy navigation between bookmarks.
