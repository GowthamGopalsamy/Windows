This sample illustrates how to clone and merge Word documents using Essential DocIO. The cloning and merging feature allows to combine the sections or contents of many documents into a single document.

Below is the illustration of the output document.

Features :

  • Combine sections from several documents into a single document.
  • Ability to clone sections, tables, paragraphs, etc.,.
  • Ability to import Contents from one document to another with options similar to Microsoft Word Copy paste functionality.
