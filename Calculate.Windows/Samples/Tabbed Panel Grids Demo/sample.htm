This sample illustrates the several aspects of working with a CalcWorkbook, which holds a collection of CalcSheet objects. The CalcSheet objects provide an ICalcData implementation for a rectangular collection of cells. These cells can be referenced using an Excel-like notation, A1, B13, and AB21. Thus, a CalcSheet object can recognize formulas such as =A1 + B3 or =Sum(A1:D5).
