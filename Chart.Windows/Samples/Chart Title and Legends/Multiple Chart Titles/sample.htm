This sample illustrates chart titles in Essential Chart.

Features:

Chart Title
  • Chart titles are created and added to a chart using ChartTitle function.
  • Multiple titles can be added to a chart. Each of the chart titles can be docked at the top, bottom, left, right, or they can be floated.
  • Chart titles can be visible or hidden by using the Visible property.
  • Font Settings- The Font and ForeColor properties are used to customize a chart title`s font.
  • Border settings- The border of a chart title can be displayed or hidden using the ShowBorder property.
