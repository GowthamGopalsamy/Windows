Essential Chart offers extensive customization for legends. The position of a legend as well as its representation aspects can be completely customized. The default legend items can also be customized through events.

Features:

  • Back Interior- The back interior of the legend can be set with gradient colors.
  • Legend Title- The legend title can be customized with the Text, ForeColor, and TextAlignment properties.
  • Shadow Settings- A legend item can display a shadow for the item`s text and icon.
  • Item Position- Items inside the legend can be aligned with various properties, such as RowsCount, Spacing, ItemsAlignment, ItemsTextAlignment.
  • Border settings- The border of a legend can be shown or hidden using the ShowBorder property.
  • Font Settings- A legend`s font can be set by the Font property. The text color of each item in the legend can be set by the TextColor property; this happens by retrieving items from the ChartLegendItem collection.
  • Item Representation- The ShowSymbol property is used to show a symbol rendered in a series and in the legend icon as well.Using the Visible property of the legend item, individual items can be hidden.
  • Custom Items and Images- Custom items can be added to the legend using ChartLegendItem class.
  • Disallow the Chart Legend- The chart control supports preventing the moving of chart legends, toolbars, and titles by utilizing the Behavior property.
