This sample demonstrates how multiple legends are used in a chart control.  It also shows how a specific series can be associated with specific legends.

Features:

    1. Creating a new chart legend- You can create a new legend and initialize it by giving it text and a name;this is done using the Name and Text properties.
    2. Customizing a Legend- Various properties are available that will customize the appearance of a legend and the properties are Position, Font, ForeColor, BackInterior, RepresentationType, ShowBorder.
    3. Legend Placement- A legend can be placed outside or within a chart area by using the LegendsPlacement property
