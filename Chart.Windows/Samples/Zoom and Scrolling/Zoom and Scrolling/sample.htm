This sample demonstrates the zooming and scrolling features in Essential Chart.

Features:

  • Zooming- Zooming is enabled along a specific axis by using the EnableXZooming and EnableYZooming properties.
    1. Zoom Button- The zoom buttons, appearing to the left and the top of the vertical and horizontal scrollbars, can be made visible or hidden by setting the ZoomButton.Size property.
    2. Zooming Factor- To produce a fine-grained range while zooming, you can set the value by which intervals are multiplied. This is done by setting the ZoomFactorX and ZoomFactorY properties.
    3. Zooming Keys- The shortcut keys used to control zooming in and zooming out are set by the ZoomIn and ZoomOut properties.
    4. Zooming Interior- The zoom area that appears while a user zooms in on a chart can be set with gradient colors.
    5. Zoom-Area Border- The zoom-area border can be made visible by setting the ShowBorder property.
    6. ZoomType- There are 3 types of Zooming.
  • Scrolling- The scroll bars of a chart control can be made visible by setting the ShowScrollBars property.
    1. Scrolling Keys- The keys used to scroll a chart area to the right, left, up, and down
  • Panning- The chart windows support a panning feature after zooming, this is done by setting the MouseAction and ZoomAction properties.
