This sample illustrates how to simulate a drill-down chart by reinitializing the chart with two different sets of data.

Features

Chart Drill Down can be achieved using the below events.

  • ChartRegionClick- The ChartRegionClick event is used to get the chart region point. Accordingly, chart data and chart type will change.
  • ChartRegionMouseHover-  The ChartRegionMouseHover event is used to set the ToolTip of the column chart series when the pointer hovers over the series points.
