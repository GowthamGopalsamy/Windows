This sample shows how to implement an interactive cursor using Essential Chart. Interactive cursors can be moved from one data point to another by dragging or move to chart area by specifying the values in XInterval and YInterval.

Features:

  • Cursor Color- The color of an interactive cursor can be set to any system-defined color.
  • MoveTo ChartArea- An Interactive curosr can be moved to chart area by using MoveToChartArea property of cursor.
  • XInterval- Cursor on the X Axis can be moved from left to right or right to left based on the value provided in XInterval property of interactive cursor.
  • YInterval- Cursor on the Y Axis can be moved from top to bottom or bottom to top based on the value provided in YInterval property of interactive cursor.
  • Moving the Cursor- The VerticalMove and HorizontalMove methods are used to move a cursor to the next closest data point in a series.
