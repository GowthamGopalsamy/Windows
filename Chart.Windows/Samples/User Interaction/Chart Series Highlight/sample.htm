This sample demonstrates the Series Highlighting feature of Essential Chart.

Features:

  • Series Highlight- To highlight a series as a whole, the chart control has a property called SeriesHighlight which must be set to true. The series can also be highlighted by hovering the mouse on particular series in the legend. The transparency can be controlled by using DimmedInterior property of the series.
  • Series Highlight Color- The highlight color for any series can be changed by adjusting the color in the HighlightInterior property.
  • Hidden Series Color- While mouse hovering on particular series all the other series in the chart will be set to the color given in DimmedInterior property.
  • SeriesHighlightIndex:-If you want to highlight only a particular series alone then you can specify the value for this property. By default it will be -1.
