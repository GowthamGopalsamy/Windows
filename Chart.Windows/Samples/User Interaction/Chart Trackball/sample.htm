This sample demonstrates how to use Trackball in Chart.

Features:

  • Trackball- Trackball can be enabled in a chart control using the Visible property of Trackball. Trackball displays information about the data point close to the current mouse location. The closest point can also be highlighted with a symbol or marker. The FormatTrackballTooltip event can be used to customize the tooltip of trackball and the Highlighting event can be used to customize how the data point should be highlighted.
