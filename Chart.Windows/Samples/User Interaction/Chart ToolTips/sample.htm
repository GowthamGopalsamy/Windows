This sample demonstrates how to create and customize ToolTips for Essential Chart.

Features:

  • ToolTips- ToolTips can be enabled in a chart control using the ShowToolTip property. By enabling this property, ToolTips associated with all the chart regions, such as Chart, Chart Area, Chart Series, and ToolBar, will be visible when the pointer hovers on the corresponding region.
  • Fancy ToolTip- A fancy ToolTip, as the name implies, displays the ToolTip in an attractive manner, which is different from other default ToolTips. The appearance of fancy ToolTips can be customized.
