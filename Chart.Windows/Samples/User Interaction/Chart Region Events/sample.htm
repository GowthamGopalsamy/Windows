This sample shows how to use chart-region events, raised by a chart control, that were started via user interactions.

Features

    The ChartRegionMouseEventHandler is used to raise chart-region mouse events such as the following.

    1. ChartRegionClick
    2. ChartRegionMouseDown
    3. ChartRegionMouseUp
    4. ChartRegionMouseEnter
    5. ChartRegionMouseHover
    6. ChartRegionMouseLeave
    7. ChartRegionMouseMove
