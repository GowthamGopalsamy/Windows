Using a toolbar makes certain actions easier, like saving a chart as an image, copying it as an image, changing styles, printing, and changing chart types.

Features:

  • Toolbar Visibility- A toolbar can be displayed by using the ShowToolBar property.
  • Toolbar Grip- A toolbar grip is used to drag a toolbar. It is made visible by using the ShowGrip property.
  • Toolbar Border- The border of a toolbar can be made visible by setting the ShowBorder property.
  • Dock-Free Toolbar- A toolbar can be docked by using the DockingFree property. It can be made dock-free by setting the property as true.
  • Toolbar Position- A toolbar can be positioned on top of a chart by using the Position property.
  • Toolbar Header- A toolbar header is made visible by setting the header size.
  • Spacing- The spacing between items of a toolbar is set using the Spacing property.
  • Disallow the Chart Toolbar- The chart control supports preventing the moving of a chart toolbar by using the Behavior property.
  • Toolbar Zoom-In, Zoom-Out, and Panning behavior- The chart control allows zooming in, zooming out, and panning using toolbar icons.
