This sample demonstrates loading and displaying chart data from a database.

Features:

  • Chart Data-Bind Model- Essential Chart exposes a flexible data model, ChartDataBindModel, that can easily display data from any source (file, database, real time).
  • The .mdb file is first converted to a data set; ChartDataBindModel then uses Database as a source for populating data using OleDbConnection, OleDbCommand, OleDbDataReader.
