The samples under Statistics show how to perform statistical analysis using Essential Chart.

Features:

  1. This sample shows the result of basic statistical functions, ANOVA, t-test, z-test, and f-test, performed between a sample series. The code below shows an ANOVA test using the BasicStatisticalFormulas class.
  2. Basic statistical functions such as mean, median, standard deviation, variance, correlation coefficient, and covariance can be calculated using the BasicStatisticalFormulas class of Essential Chart.
  3. Normal distribution for a histogram chart can be drawn by enabling the ShowNormalDistribution property.
  4. The number of histogram intervals drawn is set using the NumberOfIntervals property.
