This sample illustrates the various types of area charts supported by Essential Chart.

Features

  • Area Chart Types - There are four types of area charts you can use Area, StepArea, SplineArea, StackingArea.
  • Series Color Settings - Background and foreground colors for area charts are customized through the Interior property of the ChartStyleInfo class.
  • Chart 3-D Mode - A chart is rendered in 3-D mode by enabling the Series3D property.
  • Border settings - Border color and border width of an area chart can be changed through the Border.Color and Border.Width properties.
