This sample illustrates a box-and-whisker chart.

Features

  • Box-and-Whisker Chart- The box-and-whisker chart type is set through ChartSeriesType.BoxAndWhisker.
  • Percentile Mode:- This property is used to indicate whether the box-and-whisker chart is rendered in percentile mode or normal mode.
  • Percentile:- This property is used to set the percentile value, which should be lie between 0.0 to 0.25.
  • OutLierWidth:- This property is used to set the width of the outlier. This value should be greater than zero.
