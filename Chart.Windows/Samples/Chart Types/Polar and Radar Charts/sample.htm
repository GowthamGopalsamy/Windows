This sample demonstrates polar and radar charts created by Essential Chart.

Features:

  • Polar and Radar Chart Types- Polar and radar charts are set using the ChartSeriesType.Polar and ChartSeriesType.Radar.
  • Draw-Type- The draw-type of a chart can be changed using the RadarItem property of ConfigItems.
  • Radar Axis Style - The style of a radar axis can be set using the RadarStyle property of a chart control.
  • Inversed Radar and Polar Chart- Essential Chart supports Inversed Radar and Polar charts. This feature used to render the Polar and Radar charts in Clockwise direction.
  • Polar and Radar line style customization- Chart control supports to customize the solid circle lines of the Polar and Radar chart. Primary X and Y axes pens property is used to set the style of the solid circles.
