This sample illustrates column, column-range, and stacked-column charts.

Features

  • Column Charts- Various column charts can be applied to a chart control by using ChartSeriesType.Column, ChartSeriesType.ColumnRange and ChartSeriesType.StackingColumn.
  • Column-Width Mode- There are three column-width calculation modes that can be set through the ColumnWidthMode property: DefaultWidthMode, FixedWidthMode, and RelativeWidthMode.
  • DefaultWidthMode- The width of the columns will always be calculated to fill the space between the columns.
  • FixedWidthMode- The Width values can be given in measure units of X-axis. this.chartControl1.ColumnWidthMode = ChartColumnWidthMode.FixedWidthMode;
  • RelativeWidthMode- width of the columns are relative to the range of X-axis. Width always corresponds to 1.0 on range.
  • Column Spacing- Spacing between columns of different series is specified through the Spacing property.
  • 3D Style- The 3D chart can be render with effective color in 3D walls by enabling the Style3D property.
