This sample demonstrates how a pie chart can be implemented using Essential Chart.

Features

  • Optimize Pie Point Positions- When the OptimizePiePointPositions property is set to true, the pie chart is optimized for small values. Data points with smaller values are grouped together and ordered.
  • Show Ticks- The ticks of a pie chart are turned on using the ShowTicks property of the ChartSeries class.
  • Explode Slices- All the slices of a pie chart can be exploded (separated from the pie) with the specified explosion offset when the ExplodedAll property is enabled.
  • Doughnut structure- A pie chart can be rendered to a doughnut structure by making use of the ChartConfigItem.DoughnutCoeficient property.
  • Rotating Pie Chart- A pie chart can be rotated using the AngleOffset property.
