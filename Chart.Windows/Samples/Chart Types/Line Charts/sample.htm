This samples illustrates line, step line, spline, and rotated spline charts.

Features

  • Line Chart Types- There are four types of line charts you can make use of like line, spline, stepline, rotatedspline.
  • Border Settings- Border width and border style of a line chart are customized through the Width and DashStyle properties.
  • Point Color Settings- Data points in a series can be set to different colors by listening to the PrepareStyle event.
