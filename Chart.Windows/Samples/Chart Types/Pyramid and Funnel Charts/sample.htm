This sample illustrates pyramid and funnel charts, also known as accumulation charts.

Features:

  • Chart 3-D Mode- A chart can be viewed in 3-D mode by enabling the Series3D property.
  • Label Style- The LabelStyle property is used to customize the position of data-point labels.
  • Label Placement- The LabelPlacement property is used to customize the alignment of a label when rendered inside a pyramid or funnel chart.
  • Gap Ratio- To control the gap between points, the GapRatio property can be used.
  • Figure Base- The FigureBase property is used to allow the end user to render the base of the pyramid or funnel chart in different modes.
