This sample illustrates candle, high-low, open-high-low-close, Kagi, point-and-figure, Renko, and three-line-break charts.

Features

Financial data will usually contain more than one y-value; therefore, more than two values are added to the points of a series in a chart control.

  • Financial Chart Types- Various financial charts are applied to a chart control by using ChartSeriesType.Candle, ChartSeriesType.HiLo etc
  • Chart 3-D Mode- A chart can be rendered in 3-D by enabling the Series3D property.
  • Open-Close-Draw Mode- The OpenCloseDrawMode property will set the chart series in open, close, or both modes. This property is applicable to open-high-low-close charts.
