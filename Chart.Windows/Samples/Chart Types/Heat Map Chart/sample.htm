A heat map chart is a graphical representation of data where the values taken by a variable in a two-dimensional map are represented as colors.

Features:

  • Heat Map Style- Use Heat Map Style to set the distinct types of heat maps. The styles are Rectangular, Vertical, and Horizontal.
  • Display Color Swatch- This property is used to disable and enable the color swatch of the heat map.
  • Display Title- This property is used to enable or disable the series title in the left corner of the swatch.
  • Start and End Text- These properties are used to set the left and right labels in the color swatch.
  • Lowest, Highest, and Middle Value Color- These properties are used to set the lowest, highest, and middle color of the heat map chart; they can also be used to set the left, right, and center color of the swatch.
  • Label Margin- This property is used to set the margin in the left- and right-side labels in the color swatch.
