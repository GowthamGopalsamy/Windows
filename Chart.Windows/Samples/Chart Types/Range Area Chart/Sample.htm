This sample illustrates the range area chart type supported by Essential Chart.

Features:

  • Range Area Chart- A range area chart type can set using ChartSeriesType.RangeArea.
  • Series Points- A range area chart requires one x value and two y values.
  • Chart Custom Points- Chart custom points are used to set custom points in a series. The custom points used in this sample have images and text to represent series-point data.
