This sample illustrates bar and stacked bar charts.

Features:

  • Bar Chart Types-There are two types of area charts you can use Bar, StackingBar.
  • Chart 3-D Mode- A chart is rendered in 3-D mode by enabling the Series3D property.
  • Chart Series Points- Chart series points are displayed by using the DisplayText property in a chart control.
  • Series Color Settings- Chart series colors can be changed by using the Interior property in a chart control.
