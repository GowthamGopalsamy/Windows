This sample shows the different ways to export a chart using Essential Chart.

Features:

  • Export as an image- Chart data can be exported as an image. code
  • Export to Document- Chart data can be exported to a document by using Syncfusion DocIO.
  • Export to PDF- Chart data can be exported to a PDF.
  • Export to Excel- Chart data can be exported to an Excel sheet using Syncfusion XlsIO.
  • Export to Grid- Chart data can be exported to a grid.
  • Export to CSV format- Chart data can be exported to a CSV format.
