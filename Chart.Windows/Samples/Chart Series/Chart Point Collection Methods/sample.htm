This sample shows how to find the minimum and maximum values from a chart control series collection.

Features

  • Find Value

    • FindValue( Double)-This method returns the first chart point in the collection that has a specified first Y-value. The search always starts at the beginning of the collection.
    • FindValue (Double, String)- This method returns the first chart point in the collection that has a specified X- or Y-value.

  • FindMaxValue

    • FindMaxValue()- This method returns the first chart point in the collection with a maximum first Y-value. The search always starts at the beginning of the collection.
    • FindMaxValue(String)- This method returns the first chart point in the collection with a maximum first X- or Y-value. The search always starts at the beginning of the collection.

  • FindMinValue

    • FindMinValue()- This method returns the first chart point in the collection with a minimum first Y-value. The search always starts at the beginning of the collection.
    • FindMinValue (String)- This method returns the first chart point in the collection with a minimum first X- or Y-value. The search always starts at the beginning of the collection.
