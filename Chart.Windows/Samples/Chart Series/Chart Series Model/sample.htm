This sample demonstrates the implementation of a custom chart series model and custom chart series indexed model.

Features:.
  • Indexed Mode- Indexed models do not have an associated x-axis value for their points. The x-axis value is the position of data within a series.
  • Custom Series Model- The CustomSeriesModel class that implements the IChartSeriesModel interface provides both the "X" and "Y" values for plotting chart points.
  • Custom Indexed Series Model- The CustomIndexedSeriesModel class implements the IChartSeriesIndexedModel interface and is used in special cases where the "X" values are not needed.
  • Label Model The label for a chart series is added by customizing IChartAxisLabelModel.
