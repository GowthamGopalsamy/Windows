Custom points can be plotted on a chart to annotate interesting data. A custom point can display text, backgrounds, borders, symbols, and markers. This sample illustrates both custom points and empty points in a chart control.

Features:

  • Custom Points- Custom points can be created using ChartCustomPoint.
  • Custom Point Position- There are various options that can be used to set the position of a custom point on a chart; this is performed using the CustomType property.
    The various ChartCustomPointType options are Percent, Pixel, Chart Coordinates, Point Follow.
  • Empty Points- A chart point can be made into an empty point by setting the IsEmpty property.
  • Symbol Border- The chart control supports setting a border style for ChartStyleInfo.Symbol.
