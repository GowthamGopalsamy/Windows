This sample demonstrates different types of Trendlines available in Essential Chart.

Features

  • Visible- The visibility of trendline can be set.It is a boolean property.
  • LineStyle- The trendline line style is customized using the LineStyle property.
  • LineWidth- The width of the trendline is customized using this property.
  • Color - The color of the trendline is customized using this property.
  • Types - The different types of trendlines is selected. Default type is Linear.
  • Forward Forecast - The future of the trend is predicted using the forward forecast.
  • Backward Forecast - The past of the trend is predicted using the backward forecast.
  • Polynomial Order - The order of the change in polynomial trendline is modified.
  • Period - The period for which the moving average trendline is calculated.
