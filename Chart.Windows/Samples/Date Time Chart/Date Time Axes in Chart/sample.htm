This sample demonstrates date handling in Essential Chart. The chart series data can be defined with date-time values. The primary x-axis and y-axis of the control can be set with date-time values.

Features:

  1. Date-Time Value Type- The data type for the axis value on the chart is set to date-time using the ValueType property.
  2. Date-Time Range and Interval- The date-time range of the axis is set using the DateTimeRange property. The date-time interval is set using the axis IntervalType property.
  3. Date-Time Format- The date-time format is set by using the DateTimeFormat property.
