Essential Chart lets you customize context menus and static text in chart to specific culture.

Features:

This sample illustrates the localization of the chart context menu and static text in toolbar items. In this sample, the chart context menu is localized for the specific culture.

Note:

  • Resource file for specific culture is loaded to bin folder in the following location: ..\bin\debug\.
  • The Localize property of the chartcontrol is set to specific culture.
  • Chart control supports localization for Context Menus and static text in toolbar items.
  • Translation is executed based on tools found online; Syncfusion does not guarantee the accuracy of the translation.
