This sample demonstrates loading and displaying chart data from a database. It also illustrates how to bind axis labels in pie, funnel, and pyramid charts.

Features:

  • Chart Data-Bind Model- Essential Chart exposes a flexible data model, ChartDataBindModel, that can easily display data from any source (file, database, real time).
    The .mdb file is first converted to a data set; ChartDataBindModel then uses this as a source for populating data.
  • Chart Data-Bind Axis Label Model- The labels for the x-axis are loaded from the database using an implementation of the ChartDataBindAxisLabelModel class.
