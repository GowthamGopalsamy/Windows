This sample illustrates the various customization options for a chart control's axis tick marks.

Features:

  • Tick Label Grid-The axis-label grid is made visible by the DrawTickLabelGrid property.
  • Tick Label Grid Padding -Padding of the tick-label grid is set using the TickLabelGridPadding property.
  • Tick Color -The color of the ticks rendered on an axis is set using the TickColor property.
  • Tick size -Major tick size of the axis is set using the TickSize property, as shown in the following.
  • Small Ticks Per Interval -The number of small ticks displayed per major interval is set using the SmallTicksPerInterval property.
  • Small Tick Size -The size of small ticks displayed on the axis is set using the SmallTickSize property. By default, small ticks are not visible.
  • Tick Drawing Operation Mode -To set the tick-drawing operation mode while the chart is zoomed, the TickDrawingOperationMode property can be used.
