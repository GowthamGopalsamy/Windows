This sample illustrates various customization options for chart axis labels.

Features:

  • Text Formatting
    • Label Format- The axis labels that need to be displayed can be of various types, such as double, date-time, logarithmic, or custom.
    • Font Settings- Font settings for the primary axis can be set separately with the Font property
  • Custom Label Text
    • Custom Intervals- The labels of a chart axis can be customized by setting the TickLabelsDrawingMode property to UserMode and adding a new set of labels to the Labels collection using the ChartAxisLabel class.
  • Intersecting Labels-
    • Label Intersect Action- Labels intersect on an axis if they are too close to each other. Several options, such as Multiple Rows, Wrap, Rotate, and so on, are available to enhance the displaying of labels
    • Edge Labels Drawing Mode- Using the EdgeLabelsDrawingMode property, labels that are rendered at the edge of an axis can be adjusted.
    • Hide Partial Labels- Partially visible labels are made invisible by using the HidePartialLabels property.
