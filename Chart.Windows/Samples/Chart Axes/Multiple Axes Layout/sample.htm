This sample demonstrates the usage of a multiple axes layout in Essential Chart.

Features:

  • Multiple Axes -Multiple axes are created in the chart using the secondary axis.
  • Layout Mode-The layout mode of an axis is set using the YAxesLayoutMode property. The chart control supports combining both stacking and side-by-side chart axes layouts when multiple axes are used.All axes with the same orientation must be added to ChartAxisLayout  and PrimaryAxis.
