This sample demonstrates the crossing of axis in chart control.

Features:

  • Axis Crossing-An axis can be crossed using the Crossing property so that its points render from right to left, if horizontal; and top to bottom, if vertical. The Opposed Position property sets the axis on the opposite side of the primary axis.
  • Value Range-The value provided to Crossing property must be within the range of the x- and y-axes.
  • Primary Axes-The Crossing property is applied only to the primary axes of a chart control when a chart contains multiple axes. The crossing values must be doubled because the crossing property supports only doubled values.
