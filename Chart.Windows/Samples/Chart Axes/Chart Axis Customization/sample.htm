This sample demonstrates the customization of axis in chart control.

Features:

  • Axis Position-An axis can be reversed using the Inversed property. The OpposedPosition property sets the axis on the opposite side of the primary axis
  • Dash Style-The Dash Style property is used to set the style of a line.
  • Line Width-Width of the axis lines and axis grid lines are set using the code below.
  • Foreground Color-Foreground color of the axis lines and axis grid lines can be set separately using the LineType.ForeColor property and the GridLineType.ForeColor property.
  • Axes Titles-The TitleColor and TitleFont properties are used to set the color and font of axes titles.
  • Axes Label Placement-The AxisLabelPlacement property of axis is used to set the label position in axis.It can be placed inside or outside the plotted chart area using ChartPlacement enum.
