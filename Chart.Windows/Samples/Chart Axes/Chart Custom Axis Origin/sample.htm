This sample shows a custom origin used for a chart axis in Essential Chart.

Features:

  • Custom Origin- A custom origin can be implemented for both the x-axis and y-axis.  A custom origin is turned on through the CustomOrigin property of the ChartAxis class.
  • Origin Customization- The RangeType property is used to explicitly set custom bounds and intervals and  an axis-padding value can be calculated using the RangePaddingType property.
  • Inversed Axis- If the Inversed property of the ChartAxis class is set, then the corresponding axis range will be inverted (right-to-left or top-to-bottom).
  • Opposed Axis- An axis can be rendered on the opposite side of its usual position by setting the OpposedPosition property in the ChartAxis class.
