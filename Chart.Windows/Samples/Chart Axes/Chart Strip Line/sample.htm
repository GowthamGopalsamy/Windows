Strip lines are background bands rendered in a chart area. This sample demonstrates the implementation of strip lines in Essential Chart.

Features:

  • Strip Line Axis - A strip line can be added to a chart control's primary axis.
  • Strip Line Customization - Various appearance settings for a strip line can be customized. The strip line used to mark Historical High Temp can be customized.
  • Strip Line Position - A strip line is configured by setting its start, end, period, and width with the same value-type as the axis that holds it.
  • Strip Line Width - Sets the width of each strip line by Width property and we can set Strip Lines with a Primary X-Axis of the Date-Time Type.
  • Start at Axis Position-Sets whether or not the strip line will start at the beginning of the axis range.
  • Date Offset - Sets the offset of a strip line if the primary x-axis is of the date-time. type and StartAtAxisPosition is set to true.
  • Width Date - Sets the width of each strip line as a time span.
  • Period Date - Sets the period over which the strip line will appear.
  • Fixed-Width Strip Line - The chart control supports setting a fixed width for strip lines, irrespective of the axis range. The default value of the fixed-width property is zero.
