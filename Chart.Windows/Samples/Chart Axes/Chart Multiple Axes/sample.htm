This sample demonstrates the usage of multiple axes in Essential Chart.

Features:

  • Multiple Axes-Multiple axes are created in the chart by axea property which is demonstrates in this sample.
  • Layout Mode-The layout mode of an axis is set using the YAxesLayoutMode property.
  • Axes Title Draw Mode-When the title exceeds the axis length, it is possible to display the partial text with ellipses. The Axes Title Draw mode of an axis is set using the TitleDrawMode property.
