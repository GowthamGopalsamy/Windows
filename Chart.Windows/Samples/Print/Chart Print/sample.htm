This sample demonstrates printing a chart. Previewing and printing a chart is very easy in Essential Chart.

Features:

  • Print Preview- A print-preview dialog can be opened by showDialog method.
  • Printing Chart- A chart can be printed in gray scale, color, or "Check Printer" mode.
  • Printing Toolbar The chart control supports restricting the printing of a toolbar using the PrintToolbar Boolean property. Also, a chart control can support printing a series with textures when printing in black-and-white mode.
