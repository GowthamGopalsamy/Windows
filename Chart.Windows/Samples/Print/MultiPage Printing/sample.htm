This sample demonstrates multiple page printing in Essential Chart.

Features:

  • PrintPage Event- The PrintPage event can be triggered when printing a chart. This event is used to divide the axis range and print the chart across multiple pages.
  • HasMorePages- Set the HasMorePages property as true if an additional page should be printed; otherwise, false. The default is false.
  • Axis Range- Set the axis minimum and maximum range in the PrintPage event. The PrintPage event will be triggered until the maximum exceeds the range.
  • Printing Chart- A chart can be printed in gray scale, color, or Print Preview mode.
