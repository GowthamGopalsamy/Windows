This sample shows how to customize the appearance of a chart control.

Features

  • Chart Background- Image can be set as chart control's background using the BackGround Image property, either at run time or at design time.
  • Interior Color- The interior color of the chart can be modified using by Back Interior and Chart Interior property
  • Chart Rotate-The rotation of an axis is handled by the Rotation property of the chart area.
  • Chart Font Style-Display Text- The text displayed in a series can be shown or hidden using the Display Text property and the font face of the text can be changed using the Font property of the ChartStyleInfo class.
  • Symbol Shape- The series symbol shape can set by shape property of symbol style and chart-area shadow can be made visible by enable ChartAreaShadow.
