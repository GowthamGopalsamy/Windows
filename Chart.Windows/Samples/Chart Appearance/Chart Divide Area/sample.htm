This sample shows the divide-area feature of ChartControl.

Features

This sample demonstrates Essential Chart's support for customizing pie, doughnut, funnel,and pyramid charts when the divide area feature is enabled.

  • The GetSeriesBounds() method is used to get the bounds of the divided area when ChartArea.DivideArea is set to true.
  • The ShowSeriesTitle property is used to display the series name as a title for each chart in a divided area.
  • The ieWithSameRadius property is used to draw the pie series with the same radius when the divide area is enabled.
