This sample illustrates the styles like Grid Lines, BackColor, DashStyle, ForeColor, PenType and the Width in the PivotChart.

Code invoke to set the style for PivotChart

        this.pivotChart1.PrimaryXAxis.GridLineType.DashStyle = DashStyle.Dash;
        this.pivotChart1.PrimaryXAxis.GridLineType.ForeColor = Color.Green;
