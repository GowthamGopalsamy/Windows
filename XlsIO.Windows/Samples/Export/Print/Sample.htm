This sample illustrates the printing support of a MS Excel document.

Essential XlsIO allows to print the Excel document to printer. Use the print method of ExcelToPDFConverter class to convert the Excel to PDF and print the PDF document to printer. This enabling user easily converts the Excel document to printer.

Features:

  • Supports printing of a Excel document to printer
  • Installation of Essential PDF and ExcelChartToImageConverter is necessary to run this sample
