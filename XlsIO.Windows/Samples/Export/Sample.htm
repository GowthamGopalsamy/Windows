Getting Started

Essential XlsIO is a library that is used to create Excel workbooks without being dependent on Microsoft Excel.

This category deals with how a spreadsheet can be created from scratch and how its values can be read from an existing spreadsheet. Here is a list of samples with a short description of each.

  • Excel To ODS - This sample demonstrates Excel to ODS export in Essential XlsIO.
  • Worksheet to Image - This sample demonstrates how to convert a worksheet to an image.
  • Worksheet to HTML - This sample demonstrates how to convert a worksheet to HTML.
  • Excel to PDF - This sample demonstrates how to convert the Excel document to PDF using Essential XlsIO.
