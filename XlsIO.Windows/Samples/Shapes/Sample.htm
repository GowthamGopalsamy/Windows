Shapes help you to draw objects on your document. You can create various shapes using AutoShapeType enumeration. Once you insert an AutoShape, you can also put a text inside it and you can move or resize them as you wish

Features:

Essential XlsIO supports creating and setting up various properties for different types of shapes.
