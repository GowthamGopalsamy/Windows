Excel 2007, Excel 2010 and Excel 2013 are known for its various styles and formatting that give it an attractive appearance. Microsoft brings up the gradient styles and a number of new colors. Essential XlsIO supports formatting cells with gradient and RGB colors. You can also set styles and formatting that are supported in Biff8 format.

This sample demonstrates Essential XlsIO's support for various styles and formatting in Excel 2007, Excel 2010 and Excel 2013 format. Both built-in styles and custom styles using the interior method IStyle are used for setting gradient styles.

Note: This can apply gradient styles for the Excel 2007, Excel 2010 and Excel 2013 format.

Features:

  • Creating a calendar using XlsIO
  • Built-in styles
  • Custom styles
  • Gradients
