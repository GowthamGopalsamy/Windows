Syncfusion Essential DICOM
Essential DICOM is a .NET library that can read standard Image format and convert it into a DICOM format. The below file types are currently supported in the DICOM Converter.
  • 1.JPEG
  • 2.BMP
  • 3.GIF
  • 4.TIFF
  • 5.EMF
  • 6.PNG
