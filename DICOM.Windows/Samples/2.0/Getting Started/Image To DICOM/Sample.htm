Image to Dicom

DICOM (Digital Imaging and Communications in Medicine) is a standard for handling, storing, printing, and transmitting information in medical imaging. This sample demonstrates the conversion of standard image format to the dicom format. The list of support image types for conversion to dicom are listed below.

Support Image Types

  • Jpeg
  • Png
  • Gif
  • Tiff
  • Bmp
  • Emf

                //Initailize the DICOM Image object.
                DICOMImage dcmImage = new DICOMImage((string)this.textBox1.Tag);
                //Save the dicom image.
                dcmImage.Save("Sample.dcm");
                //Dispose the object.
                dcmImage.Dispose();
