Diagram Events

This sample illustrates the various Diagram.EventSink's events supported by Essential Diagram.

Features

  • The diagram events are listed below:
    • NodeCollectionChanged
    • NodeCollectionChanging
    • NodeSelected
    • NodeDeselected
    • NodeClick
    • NodeDoubleClick
    • NodeMouseEnter
    • NodeMouseLeave
    • SelectionListChanged
    • MagnificationChanged
    • OriginChanged
