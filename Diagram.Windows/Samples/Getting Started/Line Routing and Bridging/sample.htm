Line Bridging and Routing

This sample demonstrates the "Line Routing" and "Line Bridging" features available in Essential Diagram.

Features

  • Line Bridging : Line bridging provides the visual effect such that the links jump over other links that are found in its way with lower ZOrder, thereby avoiding the links from intersecting each other and providing a hassle-free view to clearly state the various connections between the nodes.
  • Line Routing : When a link is drawn between two nodes, by enabling the LineRoutingEnabled property of that link and the diagram view, and if any other node is found in between them, the line will be automatically re-routed around those nodes.
